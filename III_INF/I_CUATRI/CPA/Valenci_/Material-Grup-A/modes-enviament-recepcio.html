Recordant els modes d’enviament i recepció

  1. Siga una operació d’enviament send(x,...). Si quan la funció send acaba, es pot modificar x sense que afecte a l’enviament, es tracta d’un enviament:

    1. síncron.
    2. amb buffer.
    3. bloquejant.
    4. no bloquejant.

  2. Siga una operació d’enviament send(x,...). Si quan la funció send acaba, sabem que x ha arribat al procés destinatari, es tracta d’un enviament (poden haver múltiples respostes correctes):

    1. síncron.
    2. amb buffer.
    3. bloquejant.
    4. no bloquejant.

  3. Siga una operació de recepció recv(x,...). Si quan la funció recv acaba, sabem que x conté les dades rebudes, es tracta d’una recepcíó.

    1. síncrona.
    2. amb buffer.
    3. bloquejant.
    4. no bloquejant.

  4. La funció MPI_Send fa un enviament (poden haver múltiples respostes correctes):

    1. síncron.
    2. amb buffer.
    3. pot ser síncron o amb buffer.
    4. bloquejant.
