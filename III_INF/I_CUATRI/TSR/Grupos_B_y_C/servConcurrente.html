Servidor concurrente (multi-hilo)

  • Múltiples hilos de ejecución (threads) que se ejecutan de forma solapada en el tiempo
  • Cada petición es atendida por un hilo diferente
  • Ventaja.- cada hilo se planifica por separado (si se suspende no afecta al resto del servidor)
  • Estado compartido. Todos los hilos comparten un estado global
  • Requiere mecanismos de control de concurrencia para garantizar atomicidad
    • Todos los hilos comparten un estado global
    • Hay que evitar interferencias e interbloqueos
    • Inconveniente.- control concurrencia -> suspensión. Sobrecarga
    • Inconveniente.- programación y depuración complejas
  • Usado en Java y .NET

Ejemplo: servidor Echo (Servidor concurrente en Java)

import java.io.BufferedReader;
import java.io.InputStreamReader;
import java.io.PrintStream;
import java.net.ServerSocket;
import java.net.Socket;

public class EchoServer {
    public static void main(String args[]){
        ServerSocket server = null;
        Socket client  = null;
        try{
            server  = new ServerSocket(9999);
            while(true){
                System.out.println ("server listening on ..");
                client = server.accept();
                System.out.println("new client connected");
                new ClientHandler(client).start();
            }
        } catch(Exception e) { System.out.println(e); }
    }
}

class ClientHandler extends Thread {
    private Socket client = null;
    public ClientHandler(Socket client) {this.client = client;}

    public void run(){
        String buf = null;
        BufferedReader in = null;
        PrintStream out  = null;
        try{
            in = new BufferedReader(new InputStreamReader(this.client.getInputStream()));
            out = new PrintStream(this.client.getOutputStream());
            while(true){
                buf = in.readLine();
                System.out.println("Recv: "+buf);
                out.println("Echo: " + buf);
            }
            System.out.println("connection from client closed");
            this.client.close();
        } catch (Exception e){ System.out.println(e); }
    }
}
